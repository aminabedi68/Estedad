Estedad is an Arabic-Latin Sans-Serif typeface in 9 standard weights and a variable version, published for the first time in 2017 on github.

this typeface has wide codepoint range support for most Arabic and Latin languages with considerable opentype features support. design of the type is simple, smooth, compact, legible, low contrast, low optical size and optimized for web-like environments.

To contribute, see github page.
